# Thon Bunleng - Portfolio

A personal portfolio website built with React showcasing my projects, skills, and experience.

## 🚀 Live Demo

The portfolio includes 6 main sections:
- **Home** - Introduction with typewriter effect
- **About** - Technical skills and GitHub activity
- **Projects** - Recent works and applications
- **Resume** - Professional experience
- **Footer** - Contact information

## 🛠️ Tech Stack

**Frontend:**
- React 17
- React Bootstrap
- React Router DOM
- React Icons
- TypeWriter Effect

**Technologies:**
- Flutter, Kotlin, Swift

## 📦 Installation

```bash
git clone https://github.com/THONBUNLENG/portfolio.git
cd portfolio
npm install
npm start
```

## 📄 Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Removes the build dependency
